# ProductRegistration
Inventory Data Validator
Description
This project is a simple, error-proof Python script designed for basic inventory tracking. The main objective is to collect essential product details without allowing user typos to crash the program.

How it works
1. First, the script prompts the user to type in the name of the product.

2. Next, it enters a continuous loop to request the product's price, converting the text into a decimal number.

3. If the user inputs an invalid character, the system automatically catches the error and asks them to try again.

4. Finally, the same logical routine repeats to verify that the item's quantity is a whole number.

Status: Currently, the system is running smoothly in local environments and is validating all numeric inputs flawlessly.
